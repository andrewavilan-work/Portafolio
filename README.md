# Andrew Avilán — Frontend Engineer & Software Architecture Student

Professional portfolio built with **React**, **Three.js**, and **Framer Motion**. Optimized for high performance, smooth animations, and exceptional user experience.

[![Project Status: Active](https://img.shields.io/badge/Project%20Status-Active-brightgreen.svg)](https://github.com/andrewavilan-work/Andrew.dev)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

## 🚀 Key Features

- **🚀 High Performance:** Optimized with code splitting, lazy loading, and optimized assets to achieve top Lighthouse scores.
- **✨ Immersive UI:** 3D elements using Three.js (React Three Fiber) and fluid micro-animations with Framer Motion.
- **🌍 Multilingual:** Full support for English and Spanish with a custom context-based i18n system.
- **🌓 Dark/Light Mode:** Seamless theme switching with persistent user preference.
- **📱 Fully Responsive:** Mobile-first design architecture ensuring a premium experience on any device.

## 🛠️ Tech Stack

- **Core:** React 18, Vite.
- **Styling:** Vanilla CSS (Modular & Scalable).
- **Animation:** Framer Motion, GSAP.
- **3D Rendering:** Three.js, @react-three/fiber, @react-three/drei.
- **Deployment:** Optimized for Vercel/Netlify.

## 📦 Project Structure

```bash
src/
├── components/ # Reusable UI components and 3D scenes
├── context/    # Theme and Language state management
├── i18n/       # Translation dictionaries
├── sections/   # Individual portfolio sections (Lazy loaded)
└── styles/     # Global and modular CSS
```

## 🏗️ Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/andrewavilan-work/Andrew.dev.git
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```
